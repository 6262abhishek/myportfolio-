CSS Cheat Sheet

Your complete guide to CSS properties

CSS

Background Properties

  • background-attachment: Controls the background scroll behavior. Options: `scroll`, `fixed`.
  • background-color: Sets the background color (e.g., `#fff` or `transparent`).
  • background-image: Specifies an image to use as the background (e.g., `url('image.jpg')`).
  • background-position: Sets the starting position of the background (e.g., `center`, `top right`).
  • background-size: Controls the size of the background image (e.g., `cover`, `contain`).

Border Properties

  • border: Shorthand to set border width, style, and color.
  • border-width: Defines the width of the border (e.g., `thin`, `medium`, `thick`).
  • border-color: Sets the color of the border.
  • border-style: Specifies the style (e.g., `solid`, `dotted`, `dashed`).
  • border-radius: Adds rounded corners (e.g., `5px`, `50%`).

Box Model Properties

  • margin: Sets the outer spacing of the element (e.g., `margin: 10px 20px;`).
  • padding: Sets the inner spacing of the element (e.g., `padding: 15px;`).
  • width: Defines the width of the element.
  • height: Defines the height of the element.

Font Properties

  • font-family: Sets the font type for text (e.g., `Arial, sans-serif`).
  • font-size: Defines the size of the font (e.g., `16px`, `1em`).
  • font-weight: Controls the boldness (e.g., `normal`, `bold`, `100` to `900`).
  • font-style: Applies a style like `italic` or `normal` to the font.

Color Properties

  • color: Sets the text color (e.g., `#333`, `rgb(0,0,0)`).
  • opacity: Defines the opacity/transparency level (e.g., `0` for transparent to `1` for opaque).

Text Properties

  • text-align: Aligns text horizontally (e.g., `left`, `center`, `right`).
  • text-decoration: Adds decoration like `underline`, `overline`, or `line-through`.
  • text-transform: Transforms text to `uppercase`, `lowercase`, or `capitalize`.
  • text-shadow: Adds a shadow effect to the text (e.g., `1px 1px 2px #000`).

Transition Properties

  • transition-property: Specifies which properties to animate (e.g., `all`, `opacity`).
  • transition-duration: Defines how long the transition lasts (e.g., `0.5s`).
  • transition-timing-function: Controls the speed curve (e.g., `ease`, `linear`).
  • transition-delay: Adds a delay before the transition starts.

WebKit CSS3 Essentials

Selectors

  • El[att^="val"]: Matches attribute values beginning with "val".
  • El[att$="val"]: Matches attribute values ending with "val".
  • El[att*="val"]: Matches attribute values containing the substring "val".
  • El:not(s): Selects elements that are not of type "s".
  • El:nth-child(n): Selects the nth child element of its parent.
  • El:target: Selects the element targeted by a URL hash.

Flexible Box Model

Used to create flexible box layouts (HBox and VBox).

  • display: `-webkit-box;`
  • -webkit-box-orient: Sets orientation: `horizontal | vertical`.
  • -webkit-box-flex: Defines flex grow factor, e.g., `1`.

Box Sizing

Defines box-sizing options with padding and border inside the box.

  • -webkit-box-sizing: Sets box sizing to `content-box | border-box`.

Multiple Columns

Creates newspaper-style columns for text layout.

  • -webkit-columns: Shorthand for column properties.
  • -webkit-column-width: Sets the width of each column.
  • -webkit-column-count: Specifies the number of columns.
  • -webkit-column-gap: Defines the space (gutter) between columns.
  • -webkit-column-rule: Specifies a divider line between columns.

RGBA Colors

Defines colors with transparency using RGBA values.

  • rgba(0-255, 0-255, 0-255, 0-1): Color values with red, green, blue, and alpha (transparency) settings.
  • Example: `color: rgba(255,255,255,0.8);`

Border Radius

  • -webkit-border-radius: Sets rounded corners for all corners (e.g., `4px`).
  • -webkit-border-top-left-radius: Sets the radius for a specific corner.

Multiple Backgrounds

Allows multiple background images on a single element.

  • background: `url(image1) repeat, url(image2) no-repeat;`

Marquee

Creates scrolling text across the screen.

  • -webkit-marquee: Shorthand to set marquee properties.
  • overflow-x: `-webkit-marquee;`
  • -webkit-marquee-direction: Direction (e.g., `left`, `right`, `reverse`).
  • -webkit-marquee-repetition: Number of repetitions or `infinite`.
  • -webkit-marquee-style: Scrolling style (e.g., `scroll`, `alternate`, `none`).
  • -webkit-marquee-speed: Speed of marquee (`fast`, `normal`, `slow`).

Shadows

  • box-shadow: Adds shadow to boxes (e.g., `x-offset y-offset blur color`).
  • text-shadow: Adds shadow to text (e.g., `0 1px 5px rgba(0,0,0,0.3);`).

Animations

Defines animations for elements using WebKit keyframes.

  • -webkit-animation-name: Name of the animation keyframe.
  • -webkit-animation-duration: Duration of the animation in seconds.
  • -webkit-animation-timing-function: Speed curve (e.g., `ease-in`, `linear`).
  • -webkit-animation-delay: Delay before the animation starts.
  • -webkit-animation-iteration-count: Number of repetitions or `infinite`.
  • @-webkit-keyframes: Defines keyframes for animation steps.
  • Example: 
    @-webkit-keyframes colorShift {
  0%   { background-color: red; }
  100% { background-color: blue; }
}
#myDiv { -webkit-animation: colorShift 10s linear; }

Transitions

  • -webkit-transition: Shorthand for transition effects (e.g., `opacity 1s linear`).
  • -webkit-transition-property: Property to transition (e.g., `all`).
  • -webkit-transition-duration: Duration in seconds.
  • -webkit-transition-timing-function: Speed curve (`linear`, `ease-in`, `ease-out`).

Gradients

Defines gradient backgrounds with multiple color stops.

  • -webkit-gradient: Specifies gradient type and color stops.
  • Example: 
    -webkit-gradient(
  linear, 
  0 0, 
  0 100%, 
  from(red), color-stop(0.5, green), to(blue)
);

Background Size

  • -webkit-background-size: Sets the size of background images (`length_x length_y`).

Text Size Adjust

  • -webkit-text-size-adjust: Controls text size scaling (`percentage`, `auto`, `none`).

Text Overflow

  • text-overflow: Defines how overflowed text is displayed (e.g., `ellipsis`).
  • overflow: Needs `hidden` for text-overflow to work.
  • white-space: Needs `nowrap` for overflow.

Font Face

Allows custom fonts in CSS.

  • @font-face: Defines a custom font.
  • Example: 
    @font-face {
  font-family: 'MyFont';
  src: url('MyFont.file');
}
